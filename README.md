# MyToDo-List
Overview

This project is a command-line To-Do List application developed in Python 3 as part of my AP Computer Science Principles coursework. It was created for my AP Computer Science Principles performance task and demonstrates core programming concepts such as functions, lists, loops, conditionals, input validation, and exception handling.

The application allows users to add, view, and remove tasks through a simple interactive menu, providing an introduction to basic task management and program design.

Features:

- View all current tasks
- Add new tasks
- Remove existing tasks
- Interactive menu-driven interface
- Input validation for task removal
- Runs continuously until the user chooses to exit

Technologies Used: Python 3

Concepts Demonstrated
- Functions
- Lists
- Loops (while and for)
- Conditional statements
- Exception handling
- User input validation
- Modular program design
- Basic CRUD (Create, Read, Delete) operations

How to Run
- Install Python 3.
- Clone or download this repository.
- Open a terminal in the project folder.
- Run: python todo_list.py
- Follow the on-screen menu to manage your tasks.
- Learning Outcomes

This project strengthened my understanding of Python programming, modular program design, user interaction, and data management. It also provided experience developing a complete application for my AP Computer Science Principles coursework.
